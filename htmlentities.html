HTML Entities

HTML Entities Comuns

  • < : &lt; - Menor que

  • > : &gt; - Maior que

  • & : &amp; - E comercial

  • " : &quot; - Aspas duplas

  • ' : &apos; - Aspas simples

  • / : &#47; - Barra

Espaços e Quebras de Linha

  • (espaço) : &nbsp; - Espaço não quebrável

  • (tabulação) : &#9; - Tabulação

  • (nova linha) : &#10;

  • — : &mdash; - Traço longo

  • – : &ndash; - Traço curto

Caracteres Tipográficos

  • © : &copy; - Símbolo de copyright

  • ® : &reg; - Símbolo de marca registrada

  • ™ : &trade; - Símbolo de marca registrada

  • € : &euro; - Euro

  • £ : &pound; - Libra esterlina

  • ¥ : &yen; - Iene

Setas

  • ← : &larr; - Seta para esquerda

  • → : &rarr; - Seta para direita

  • ↑ : &uarr; - Seta para cima

  • ↓ : &darr; - Seta para baixo

Matemáticos

  • × : &times; - Multiplicão

  • ÷ : &divide; - Divisão

  • ± : &plusmn; - Mais ou menos
